# Трекер достижений
======

## Описание:
Отслеживает прогресс достижений с выводом в чат и уведомления  в центре экрана.

------

Комманда | Описание
| ------------- | ------------- |
| ach | Включение/Отключение модуля
| ach alert | Включение/Отключение: Уведомление в центре
| ach status | Состояние модуля
